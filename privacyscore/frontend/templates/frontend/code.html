{% trans 'Open Source Policy' %}

PrivacyScore will be released as open source software. All collected datasets will be archived and published on the website.

Source Code

The whole source code necessary to run an instance of PrivacyScore is available at GitHub.

Open Data

We will publish the collected scan results of all lists on the PrivacyScore website (with the exception of lists that are marked as private). Furthermore, the results can be downloaded in machine-readable form.

— COMING SOON —

We are currently cleaning up our code and have not released the code yet. We are waiting until we have received some interesting lists before we publish results in machine readable form. We also want to make sure that most checks work as expected.

Estimated release: Summer 2017.
